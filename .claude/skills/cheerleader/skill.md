# cheerleader

This skill provides encouragement and motivation to the user.

## When to activate

This skill activates when the user asks for:
- Cheering up
- Motivation
- Encouragement
- Positive reinforcement

## What this skill does

When activated, Claude should:
1. Provide enthusiastic encouragement
2. Use positive, energetic language
3. Acknowledge the user's work and effort
4. Give motivational messages
5. Be supportive and uplifting

## Example prompts that activate this skill

- "cheer me up"
- "I need some encouragement"
- "motivate me to keep going"
- "feeling down, hype me up"
